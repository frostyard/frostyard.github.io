All desktop images include a full GNOME session, Flatpak, CUPS printing, PipeWire audio, Podman with Distrobox, and a broad set of hardware firmware. They differ in kernel and bundled applications.

| Image | Kernel | Extra Applications |
|-------|--------|--------------------|
| Snow | Debian backports | — |
| Snow Loaded | Debian backports | Edge, VS Code, Bitwarden, Incus, Azure VPN |
| Snowfield | linux-surface | — |
| Snowfield Loaded | linux-surface | Edge, VS Code, Bitwarden, Incus, Azure VPN |
